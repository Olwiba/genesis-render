---
title: Blocks
description: Supported block types and their props
---

Blocks map to `@olwiba/ui` components. Each block has a `type` name, typed `props`, and optional `children` slots.

## Using a Block

```json
{
  "root": "my-block",
  "elements": {
    "my-block": {
      "type": "BlockName",
      "props": { "key": "value" },
      "children": []
    }
  }
}
```

## Registered Blocks

### PageHeader

Page title with optional description. Accepts children.

| Prop | Type | Required |
|------|------|----------|
| `title` | `string` | Yes |
| `description` | `string` | No |
| `className` | `string` | No |

### Spinner

Loading spinner.

| Prop | Type | Values |
|------|------|--------|
| `size` | `enum` | `sm` `md` `lg` |
| `className` | `string` | — |

### GlassCard

| Prop | Type | Values |
|------|------|--------|
| `blur` | `enum` | `sm` `md` `lg` |
| `className` | `string` | — |
